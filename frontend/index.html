How It Works

System Workflow of EcoPackAI

A complete pipeline from user input to AI recommendation, analytics, and sustainability reporting.

01
User Inputs

Material attributes and filters are entered from the frontend UI.

02
Flask API Processing

REST endpoints validate data and pass it to the prediction pipeline.

03
ML Prediction

Cost and CO2 predictions are generated using trained models.

04
Eco Score Ranking

Materials are scored and ranked to identify the best recommendation.

05
Analytics Summary

Dashboard metrics compare top recommendations against the baseline.

06
Report Export

Users can generate PDF and Excel sustainability reports.
